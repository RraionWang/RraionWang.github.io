# 语音助手（KWS）
现在的语音助手在我们的生活里面越来越广泛，无论是苹果的Siri还是小米的小爱同学，再到天猫精灵等等。本项目将帮助你实现一个属于自己的语音助手，先小小地期待一下吧🙂。



!!! info  "版本注意"

    请注意版本差异，参考[此文章](https://wiki.seeedstudio.com/xiao_esp32s3_sense_mic/)


## 段位
无需任何基础

## 实现功能
说yes实现led亮灯，说no实现led灭灯

## 准备
1.XIAO ESP32S3 Sense 开发板
2.microSD卡（不大于32G）
3.microSD卡读卡器
4.USB-Type-C数据线
