# 人脸识别报警器

## 段位
无需任何基础

## 实现功能
实现检测人脸的时候蜂鸣器响，人脸离开的时候蜂鸣器不响。

## 备料
1. XIAO ESP32S3
2. 一个蜂鸣器
3. type-C数据线（注意充电线可能不行哦）
