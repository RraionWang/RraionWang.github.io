# 触摸屏布局

我很喜欢使用[1.47寸的触摸屏](https://item.taobao.com/item.htm?id=825365356383&mi_id=0000d9pcTTS3OQcNj7lKwv7XwLWZ3z7UOU9t1-yxbG-XlvQ&spm=tbpc.boughtlist.suborder_itemtitle.1.75552e8dfXIMyQ)，比较小巧，价格也比较能接受。

但是在布局的时候需要注意：
1. 红框框住的地方有触摸芯片，所以此位置尽量不要布线布局（正面的时候）。
2. 在排线侧不要放置按键等输入设备，否则机械位移会让排线不断弯折
