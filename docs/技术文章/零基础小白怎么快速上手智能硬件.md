# 零基础小白怎么快速上手智能硬件？

## 导言
我想很多非技术行业的朋友，对DIY智能硬件都有一种陌生感，如果让你上手，你可能会说：
>“这个太难了，我不会做，🙅‍♂我不是写代码的”。

那我们作为创客社区，如何应对🧐？

## 💡想法

技术易寻，但是想法难得。所有的小项目或者小DIY也好，如果涉及到硬件，那么无非就是只有三部分：输入，控制，输出。
