
# 小红书的故事
2025年的8月末，我离开了上一份工作，在自己的gap期，我一边看新的工作机会，一边做做自己的小项目，偶尔在社媒上发发。

故事的转机出现在11月22号，这天晚上，我随手发布了一个oiiai猫的meme帖子，没想到一觉醒来这个帖子突然火了，获得了1.7万的点赞和超10w的浏览。几百个人私信我问我有没有同款玩具想要购买，我一开始给大家回复了原玩具的链接，但是后来发现玩具有两个问题：

1. **不带喇叭**
2. **价格过于贵了**（大概要两百多块钱）

这对于大家来说是比较难接受的。所以我干脆就决定手搓一个，降低成本，并且准备换一个更大的屏幕。初步售价定在50元

在制作的过程中我才慢慢发现，对我个人而言不太可能做量产。原因主要有以下几点。

1. **物料不可控** 电子元件+屏幕+电池+外壳整体已经超过50元了，再加上邮费，所以我几乎完全是亏本的状态，更不要说还有邮费了。
2. **工厂的贴片开机费太贵** 路板上的元件需要焊接，如果选取工厂焊接的话开机费要500多块钱，太贵了我承担不起，所以最终还是使用手工焊接，而手工焊接的话做量产肯定不可能。
3. **售后不可控** 做硬件本来就有很高的边际成本，所以如果一旦出现产品质量问题，几乎只有召回这一条路，所以为了我的口碑以及可持续发展，我选择对每一个购买产品的朋友负责，建立了专属的客户文档和售后群。如下图

所以后来将定价改到了65元，尽管如此几乎还是为爱发电，但是令我开心的是，客户的反馈十分正向，尤其是一位叫【大蒜头莎莉】的网友，给了我很多的建议和意见。这是我第一次通过自己的知识储备和业余技能赚到钱。

这一次非常特殊的经历不仅让我看到了自己的可能性，也感受到了来自他人的善意，我后续还会慢慢做有趣的小东西，感谢大家的支持，希望自己可以给大家带来开心快乐。
